# Product Overview

## Product Purpose

Interview Manager (IMS) 是一款公司内部面试管理桌面客户端产品，部署为 Tauri 桌面应用。

产品核心不是单纯的聊天工具，而是一个完整的候选人工作台：
1. 从公司内部系统登录并拉取面试数据
2. 管理远程候选人和纯本地候选人
3. 导入、解析、筛选简历
4. 进入每位候选人的 AI 面试工作台
5. 共享完整候选人档案给局域网内其他面试官

核心价值：统一远程数据、本地导入数据与 AI 产物，为每位候选人建立持续的、可回溯的工作空间。

## Target Users

- **技术面试官** - 需要快速了解候选人背景、进行 AI 辅助面试
- **招聘负责人** - 需要管理候选人池、跟进面试进度
- **招聘协调同学** - 需要导入简历、协调面试安排

## Key Features

1. **登录与 Token 生命周期管理** - 公司内部网页登录，本地 token 安全存储，过期自动提示重新验证
2. **候选人管理** - 远程数据同步、本地候选人创建、主列表搜索与筛选
3. **简历导入流水线** - 支持 zip/pdf/图片批量导入，OCR 文本提取，结构化解析，候选人自动归并
4. **AI 工作台** - 每位候选人独立 OpenCode workspace，反馈驱动产物版本化
5. **候选人档案共享** - IMR 格式打包，局域网在线发送或离线导出导入

## Business Objectives

- 让简历导入、初筛、出题、评价、共享形成闭环
- 保持本地优先，降低对远程系统能力的依赖
- 保持候选人档案的持续可回溯性

## Success Metrics

- 搜索响应 < 500ms
- 主列表支持 1000+ 候选人浏览
- 导入处理不阻塞 UI

## Product Principles

1. **本地优先** - 核心功能不依赖远程系统，本地数据闭环
2. **Token 安全** - 只存 OS Keyring，不落明文
3. **反馈驱动** - AI 产物不直接手工编辑，通过反馈驱动 agent 生成新版本
4. **候选人独立空间** - 每位候选人独立 workspace，数据隔离

## Technical Stack

| 组件 | 技术 |
|------|------|
| 桌面壳 | Tauri v2 |
| 本地服务 | Bun.serve |
| 数据库 | bun:sqlite + Drizzle ORM |
| 前端 | Vue 3 + Vite + Tailwind |
| AI 引擎 | 内置 opencode-ai |
| 包管理 | pnpm workspace |
