## 根本性问题汇总

### 1. 技术栈错误（Phase 3）

**原版声称**: LangGraph  
**代码库实际**: DeepAgents（`docs/Agent-Extension-Architecture.md:45` 明确拒绝 LangGraph）  
**影响**: phase-3-detailed.md 整个技术方案需重写

### 2. 前端架构错误（Phase 1.1）

**原版声称**: `defineStore('message')` + `messageApi`  
**代码库实际**: `createLuiMessageModule` 工厂 + `luiApi`  
**影响**: phase-1-detailed.md 前端集成部分需重写

### 3. 后端架构错误（Phase 1.1）

**原版声称**: 新建 `MessageService` + `routes/messages.ts`  
**代码库实际**: 消息逻辑在 `routes.ts:2210`，无独立 Service  
**影响**: phase-1-detailed.md 后端部分需重写

### 4. Bug 状态错误

**原版声称**: candidate-selector 有 Bug 待修复  
**代码库实际**: `candidate-selector.vue:22-24` 已正确显示姓名  
**影响**: 删除该修复任务

### 5. reasoning 功能不适用

**原版声称**: AgentChat 需添加 reasoning 展开/收起  
**代码库实际**: 当前 AI Gateway 不输出 reasoning 字段，AgentChat.vue 无 Collapsible  
**影响**: 该任务暂缓或删除

### 6. Phase 4-6 技术方案严重缺失

**原版问题**: 仅功能描述，无 API 路由设计、无前端组件方案  
**建议**: 补充 API 路由设计和关键文件路径
